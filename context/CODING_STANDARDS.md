# Coding Standards

## Naming Conventions

| Type | Convention | Example |
|------|-------------|----------|
| Class | PascalCase | StudentDashboard |
| Method | camelCase | submitExam() |
| Variable | camelCase | studentName |
| Constant | UPPER_CASE | MAX_TIME |
| Package | lowercase | com.exam.view |
